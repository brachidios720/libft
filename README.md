# libft

Bibliothèque personnelle en C regroupant des fonctions utilitaires réécrites à partir de la libc ainsi que des fonctions supplémentaires.

## À propos

`libft` est l’un des premiers projets du cursus 42. Il sert de base pour de nombreux projets suivants.

## Contenu typique

- fonctions de manipulation de chaînes
- gestion mémoire
- conversions
- fonctions d’affichage
- listes chaînées

## Compilation

```bash
make
Utilisation
